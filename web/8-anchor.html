Anchor tag

HyperText Markup Language(HTML) is the standard markup language for creating web pages and web applications.

<a href="https://www.w3.org/html/" target="_blank" title="html">HyperText Markup Language(HTML)</a> a는 link를 걸어주는 tag
href는 어떤 레퍼런스로 하이퍼링크를 걸 것인가, target="_blank"는 새 tab에서 열리도록, title은 마우스 호버 시 뜰 툴팁 내용에 대한 attribute
